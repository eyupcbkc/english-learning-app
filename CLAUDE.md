# English Learning Project - AI Context

## Amac
Eyup'un A1'den C1'e Ingilizce ogrenme yolculugu. Web app ogrenme araci.
**Oncelik: Ogrenme > Icerik > App**

## Ogrenci Profili
- Mevcut seviye: A1 (Beginner)
- Hedef seviye: C1 (Advanced)
- Ana dil: Turkce
- Yazilimci degil - teknik detay verme, sadece calistir

## Session Protokolu
1. `docs/progress.md` oku → neredeyiz?
2. `docs/roadmap.md` oku → siradaki unite ne?
3. Sadece ilgili unite dosyasini ac

## Teknik
- React + Vite, Pure CSS, localStorage
- DB yok, backend yok
- Icerik JSON: `src/data/units/unit-XX.json`
- Yeni unite = yeni JSON + unitIndex.js'e import ekle
- GitHub: github.com/eyupcbkc
- Deploy: Vercel (ileride)
